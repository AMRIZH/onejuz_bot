OneJuz - Bot

Bismillah, OneJuz - Bot adalah aplikasi bot berbasis telegram yang bertujuan memudahkan pelaporan dan pemantauan tilawah bagi komunitas One Day One Juz. Berikut adalah spesifikasi dan cara penggunaan Bot:

1. Spesifikasi

  • Dibuat dengan Node.JS
  • Database MongoDB
  • Server Heroku (free tier)
  • versi terakhir 1.0.0-alpha.2
  • Project ini open source, repository bisa dilihat disini, kontribusi sangat dipersilakan.
Catatan: Karena menggunakan server gratis, Bot kadang akan mengalami down pada waktu-waktu tertentu. Ini bersifat sementara sampai Bot sudah masuk versi 1.0.0, nanti akan dipindahkan ke server berbayar tanpa down time.

2. Cara penggunaan

Bot ini bekerja dalam grup Telegram, dan akan membalas dengan pesan error apabila digunakan dalam personal chat. Dibawah ini adalah beberapa tata cara penggunaan dan perintah yang dapat dijalankan di bot:

  1. Untuk memasukkan bot kedalam grup, bisa dengan membuka menu add member di pengaturan grup, kemudian cari bot onejuz_bot
  2. /register - Saat pertama kali memasukkan bot kedalam grup, ketikkan perintah ini untuk mendaftarkan dirimu ke database. Ini khusus bagi user yang telah berada dalam grup ketika bot dimasukkan. Apabila ada user yang bergabung dengan grup ketika bot sudah ada, maka user tersebut langsung terdaftar.
  3. Pelaporan bacaan satuan: kamu bisa mengetikkan langsung juz yang sudah dibaca, misal 2. Atau dengan teks seperti Lapor juz 3 kholas
  4. Pelapuran bacaan multiple: Bot juga memahami pelaporan bacaan sekaligus, misal Lapor juz 2, 3, 4 5 kholas
  5. /restart ulangi bacaan - Perintah ini untuk mereset bacaan yang sudah dibaca. Misal sudah memasuki romadhon, pingin baca dari juz 1, bisa menggunakan perintah ini.
  6. /rename Nama Baru - Apabila nama yang kamu daftarkan di telegram salah, kamu bisa menggantinya dengan perintah ini. Karena secara default saat register, nama dari telegram yang akan masuk ke database.
  7. /renameg Nama Grup Baru - Perintah ini untuk mengganti nama grup jika diperlukan (admin only)
  8. /statistics - Untuk melihat laporan bacaan harian user (admin only). Setipa hari bot juga akan mengirimkan laporan pada jam 23:59, lalu laporan mingguan akan dikirm hari sabtu jam 23:59.

3. FAQ

Berikut beberapa pertanyaan yang mungkin muncul terkait bot
  • Data pribadi apa saja yang disimpan saat mendaftar? Hanya nama, username dan id telegram saja. Apabila user keluar dari grup datanya pun akan terhapus
  • Siapa pembuat bot ini? Bot ini dibuat oleh orang pribadi, tidak terkait dengan organisasi apapun
  • Apa peran admin setelah ada bot ini? Admin akan tetap berperan untuk memantau laporan dan memotivasi anggota untuk terus membaca qur'an
